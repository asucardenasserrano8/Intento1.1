# Intento1.1

App financiera para llevar control de finanzas personales en **Streamlit**.

## Funcionalidades

- Registro de ingresos y gastos por concepto.
- Resumen mensual automático (ingresos, gastos, balance y tasa de ahorro).
- Definición de meta de ahorro con barra de progreso.
- Historial de movimientos.
- Descarga del historial en CSV.

## Ejecutar localmente

```bash
pip install -r requirements.txt
streamlit run app.py
```

La app abrirá en `http://localhost:8501`.

## Deploy en Streamlit Cloud

- **Ruta del archivo principal**: `app.py`
