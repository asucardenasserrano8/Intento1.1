# Intento1.1

App financiera web para llevar control de finanzas personales.

## Funcionalidades

- Registro de ingresos y gastos por concepto.
- Resumen mensual automático (ingresos, gastos, balance y tasa de ahorro).
- Definición de meta de ahorro y barra de progreso.
- Historial de movimientos.
- Persistencia en `localStorage`.

## Ejecutar localmente

```bash
python3 -m http.server 4173
```

Después abre `http://localhost:4173`.
